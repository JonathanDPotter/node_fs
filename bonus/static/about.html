About

This is my version of the TrueCoders Node FS exercise.

This exercise is intended to help me learn and remember the uses of the Node file system module.

I am using the file system to read .html files and display them to the browser with my Node server. I also made a custom logger for the server that logs all requests to the console in the format: [METHOD] STATUS_CODE path:"PATH" TIME_OF_REQUEST.
